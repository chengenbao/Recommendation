How to install?
===============

Requirements
------------

* python 3

Package manager
---------------

An AUR package is not yet available. 

PyPI
----

`See Pypi <http://pypi.python.org/pypi/bibtexparser/>`_

To install with pip:

.. code-block:: sh

    pip install bibtexparser


Manual installation
-------------------

`Download <http://source.sciunto.org/bibtexparser/>`_

.. code-block:: sh

    python setup.py --root=/usr/local/bin
